Reproduzindo áudios

source src="Progressive Rock - Audionautix.mp3" type="audio/mpeg"

audio/mpeg é usado para arquivos mp3. se o arquivo for ogg é usado audio/ogg. e se for wav é usado audio/wav.

É aconselhável usar os 3 formatos para que atinja um número maior de navegadores

Controls possibilita que o usúario pause e despause o áudio. Além disso é aconselhável o uso de metadata no preload, pois carregará apenas informações essenciais antes de abrir o site

É possível deixar o áudio em loop usando o parâmetro loop
