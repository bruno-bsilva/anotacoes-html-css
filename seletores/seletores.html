Descobrindo os seletores

Id

Ao usar o atributo id podemos diferenciar um h1 de outro, ou melhor, podemos diferenciar qualquer campo de outro.

Um id é unico no seu documento, então o correto é só se usar uma vez.

Uma class é outro tipo de seletor. Esse seletor é ideal quando se quer selecionar mais de um elemnto na mesma classe, logo ele não é único.

Id se representa com # e class com .

É possível usar duas class em um mesmo elemnto, sendo necessário apenas separar por espaço. ex class= "primeira segunda".
