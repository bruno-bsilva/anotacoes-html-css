Familias tipograficas

Serifadas

Fontes serifadas são aquelas que têm uma perninha no fim.

Sans-Serif

Sans-Serif são fontes sem serifa.

Monoespaçadas

São fontes com espaços igualitários que uma letra ocupa.

Aplicação

Para aplicar essa fonte basta utilizar a tag font-family. Lembrando que é muito importante declarar mais de uma fonte, caso alguma delas esteja indisponivel em determinado dispositivo.
