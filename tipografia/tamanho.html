Atributos da fonte

Medida

Os sistemas de medidas recomendados são px e em. O tamanho padrão de uma fonte é 16px ou 1em logo 1em = 16px. Para mudar o tamanho de uma fonte basta utilizar a tag font-size

Peso

O peso seria a "intensidade" da fonte, sendo elas lighter, normal, bold e bolder, lembrando que uma fonte pode não possuir todos esses pesos. Para mudar o peso use a tag font-weight

Estilo

Utilizando a tag font-style é possível mudar o estilo da fonte

Shorthand

Usando o atributo font, é possível usar apenas uma linha para declarar style, weight, size e family, nessa ordem.
